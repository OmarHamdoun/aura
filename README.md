AURA — Autonomous Understanding & Reasoning Assistant

Tagline: Omnidirectional perception for robots: live captions, structured insights, simple actions.

What it is:
AURA turns webcam or fisheye video into real-time scene captions and safe, explainable navigation hints. It’s promptable, model-switchable (FastVLM / Qwen2.5-VL), and runs in a sleek Django UI with caption history and time-stamped decisions."# aura" 

Mini-InternVL2-2B-DA-DriveLM is supported when `MINI_INTERNVL2_DRIVELM_PATH` points at the local model folder (defaults to `/home/vision/work/Mini-InternVL2-2B-DA-DriveLM`).
